# 🚀 Node.js Backend Auto Deployment (Linux VM)

This project includes a fully automated setup script that installs:

- Node.js (LTS)
- Nginx reverse proxy
- Systemd service (auto start/restart)
- Git-based deployment
- Public IP / domain configuration
